夢ビジョン サーマルフュージョンプログラム

本プログラムおよび基板はナイトビジョンカメラにプリズムでサーマル映像を合成するための、LCD出力プログラムと基板である。

・処理装置：Raspberry Pi Zero2

・OS：DietPi

・LCD：ST7789 240x240

・対応サーマルカメラ
  - HIKMICRO mini2
  - Infiray P2 Pro




mini2とP2Proの違いについて

・ mini2は256x196で生画像を取得可能、256x192で正規化済の画像を取得可能

・ P2Proは256x196で正規化済の画像を取得可能、256x384で正規化済の画像+生画像を取得可能
